# CODE_OF_CONDUCT

# Living Civilization Framework (LCF)

## Code of Conduct

---

## Purpose

The Living Civilization Framework (LCF) is an open project that welcomes constructive participation from people with different backgrounds, experiences, disciplines, and perspectives.

The purpose of this Code of Conduct is to help create a respectful, collaborative, and productive environment for discussion, research, documentation, and contribution.

---

## Our Values

We encourage contributors to:

- Treat others with respect and professionalism.
- Welcome constructive discussion and different viewpoints.
- Focus on improving ideas through evidence, reasoning, and practical experience.
- Support continuous learning and collaboration.
- Recognize the dignity of every honest contribution.

---

## Expected Behavior

Contributors are encouraged to:

- Be respectful and courteous.
- Discuss ideas rather than attack individuals.
- Provide constructive feedback.
- Support claims with evidence or clear reasoning where appropriate.
- Accept that reasonable people may disagree.
- Be willing to learn, revise opinions, and improve ideas.
- Respect cultural diversity and different professional backgrounds.

---

## Unacceptable Behavior

The following behaviors are not acceptable within this project:

- Personal attacks or harassment.
- Hate speech or discriminatory behavior.
- Threats or intimidation.
- Knowingly presenting false or misleading information as established fact.
- Trolling or intentionally disruptive behavior.
- Repeated bad-faith arguments intended to prevent productive discussion.
- Violations of applicable laws or the rights of others.

---

## Constructive Discussion

LCF encourages thoughtful exploration.

Contributors should:

- Question ideas respectfully.
- Explain disagreements clearly.
- Welcome evidence that challenges existing assumptions.
- Help improve the framework rather than defend positions unnecessarily.

Constructive criticism is an important part of the project's continuous improvement philosophy.

---

## AI-Assisted Contributions

AI tools may be used to assist with:

- Brainstorming
- Drafting
- Editing
- Research assistance

Contributors remain responsible for reviewing AI-assisted content before submission.

AI-generated material should be checked for accuracy, attribution, and consistency with the goals of the project.

---

## Maintainer Responsibilities

Project maintainers are responsible for helping maintain a respectful and productive environment.

Maintainers may:

- Request clarification or revisions.
- Moderate discussions.
- Remove inappropriate content.
- Decline contributions that do not align with this Code of Conduct or the project's contribution guidelines.

These actions should be taken fairly, consistently, and in good faith.

---

## Scope

This Code of Conduct applies to:

- Repository discussions
- Issues
- Pull requests
- Project documentation
- Community interactions related to the repository

---

## Reporting Concerns

If you believe this Code of Conduct has been violated, please contact the project maintainers through the appropriate GitHub communication channels.

Reports should include sufficient information to allow the situation to be reviewed fairly.

Maintainers will make reasonable efforts to review reports objectively and, where appropriate, respond in a timely manner.

---

## Continuous Improvement

Like the Living Civilization Framework itself, this Code of Conduct may evolve over time through discussion, practical experience, and community feedback.

Constructive suggestions for improvement are welcome.
